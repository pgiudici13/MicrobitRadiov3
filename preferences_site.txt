## 2. Interaction & UX Standards
- **Magnetic Mouse**: 
  - **High Response LERP (0.36)** for cursor following.
  - **Sequential Capture**: Only ONE element can be "captured" at a time to prevent visual jitter in dense areas.
  - **Snap-In Animation**: On hover, elements have a **250ms Transition (Cubic-Bezier)** to smoothly move into the mouse position.
  - **Active-Follow Mode**: After the 250ms snap-in, the element switches to **0s Transition** for zero-lag following.
  - **Cursor Implosion**: When captured, the custom cursor shrinks (`scale(0.1)`) and fades (`opacity: 0`) rather than vanishing instantly.
- **Background Physics**:
  - Interactive lattice grid with web-line mouse connections.
- **Zero-Scroll Start**:
  - Forced `window.scrollTo(0,0)` on load and post-intro.

## 3. Architecture Visualization
- **Focus Rule**: Target internal elements for magnetic effects in complex diagrams.

## 4. Maintenance Notes
- CSS `transition: transform 0.25s` is the default for magnetic elements to allow the snap-in; JavaScript adds `.active-follow` to bypass this for realtime motion.
