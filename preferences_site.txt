## 3. GitHub API Infrastructure
- **Endpoint**: `https://api.github.com/repos/pgiudici13/MicrobitRadioV3`.
- **Initialization**: `fetchGithubData()` must be called inside the `DOMContentLoaded` listener.
- **Robustness**: 
  - Individual `res.ok` checks per fetch are mandatory.
  - Releases: If 404 (no releases), fallback to a static "v0.1.0" or "Live" tag.
  - Repo check: If the root repo fetch fails (e.g. rate limit), stay on placeholders.
